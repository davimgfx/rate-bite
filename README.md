# RateBite 👩‍🍳⭐
A RateBite é um site onde é possível cadastrar restaurantes, bem como avaliações de clientes e o preço médio dos estabelecimentos.

### Construído com:
#### Front-end
- Next.js <br>
- React.js <br>
- Tailwind.CSS  <br>
#### Back-end
- Express
- Node.js
#### Banco de dados
- PostgresSQL
